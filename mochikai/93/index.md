# Diffusion Modelとは

1. 生成モデルの一種（生成モデル＝GANとかVAEとか）
2. 学習データに対してT回ノイズ（正規分布βN(0,α), βは小さい値）を何度も足して行くとT→∞でノイズになる
3. ↑の逆変換を無敵のディープラーニングで求めることができれば、ノイズから学習データ（の分布を持つ新しい生成結果）を作れる

### pros/cons

- GANより構造的な見通しが良い
- GANより高精細画像の生成に成功している例が多い
- 計算量は基本的にGANより大きくなる
